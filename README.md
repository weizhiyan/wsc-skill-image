> 把你脑子里模糊的画面，变成 AI 能精准执行的提示词。

直接说想法就行——不用背语法，不用懂参数。wsc-image 会自动识别你的意图、引导你确认关键信息、输出经过结构化优化的专业提示词，并在每次使用后积累你的风格偏好，下次更精准。

---

## 它能做什么

### 场景 A：AI 生图提示词

你说一句话，它输出一段专业提示词。背后使用 **7 层视觉结构**，覆盖从主体到氛围的完整描述维度：

```
① 主体描述    — 核心对象是什么（人/物/场景）
② 外观细节    — 材质、纹理、颜色、形态
③ 光影设定    — 光源方向、光质、特殊光效（体积光/丁达尔/逆光）
④ 构图镜头    — 角度、景别、焦距感、景深
⑤ 环境氛围    — 背景、时间、天气、情绪
⑥ 风格化语言  — 画风、渲染方式
⑦ 平台参数    — 比例、质量、版本（按目标平台自动匹配）
```

**输入示例：**「生成一个氛围感，透视很强的海报，海报就以夏日为主题」

**输出：** 一段包含主体、光影、运镜、氛围、风格参数的完整提示词，放入代码块直接复制用。
```
3D 运营 KV 海报，夏日主题，超高清 C4D 渲染。
画面中心：一个巨大的透明玻璃质感冰块，内部包裹着一片鲜嫩的薄荷叶和柠檬片，晶莹剔透。
视觉冲击：冰块周围环绕着飞溅的水珠、明黄色的充气泳圈、蓝色的冲浪板和粉色的果冻质感字母“SUMMER”，物体呈现放射状向镜头“弹出”，具有极强的广角透视感。
材质与光影：极致的 Octane 渲染效果，具有真实的次表面散射（SSS）质感，水滴光泽莹润，色彩明亮。
配色：清新多巴胺色系（薄荷绿、柠檬黄、透明蓝、西瓜粉）。
背景：纯净的浅色渐变背景，突出主体。
画幅：3:4 比例，运营 KV 构图，极高清晰度。
```

### 场景 B：视频 / 动画分镜提示词

描述一个场景或故事，输出完整分镜结构，包含：

- 镜头设计表（分镜编号 / 画面内容 / 运镜方式 / 时长 / 光影氛围）
- 运镜词汇：推拉 / 横移 / 环绕 / 跟拍 / 高空俯瞰 / 超慢动作…
- 每个镜头的独立提示词（可直接投喂即梦视频 / 可灵 / Sora）

---

### 场景 C：参考图风格提取

上传一张图 + 说「参考这个风格」，会从四个维度拆解它：

| 维度 | 说明 |
| :--- | :--- |
| 视觉风格 | 扁平插画 / 3D渲染 / 写实摄影 / 赛博朋克… |
| 色彩调性 | 冷暖倾向 / 饱和度 / 对比度 / 主色系 |
| 光影质感 | 硬光/柔光 / 光源方向 / 体积光等特殊光效 |
| 构图比例 | 主体位置 / 留白方式 / 画幅比例 |

分析完成后强制弹窗确认你的意图（保留整体风格 / 只要质感 / 只是找灵感 / 自定义），再根据选择生成提示词。

---

### 场景 D：修图 / P图提示词

上传原图，描述你想改什么，自动判断子类型后生成提示词：

**修图模式**（不替换主体，只增强画面）
- 适用：调色、光影重构、电影感重建、皮肤质感优化
- 输出：在原图基础上增强，保留人物位置与构图不变

**P图模式**（替换/新增画面元素，保留原主体）
- 适用：换背景、换场景、换服装、保留人物换环境
- 核心机制：弹窗确认保留哪些元素（五官/姿态/服装/光影方向）
- **五维保护原则**：严格锁定五官比例、脸部骨相、肤色、身材比例、不改变年龄/种族，防止生图时人脸漂移

**多图合成 / 三宫格模式**（多张图融合或拼接输出）
- 适用：当你需要把多张参考图融合成一张，或输出竖屏三宫格布局时触发
- 输出：整体比例说明 + 每格画面描述 + 字幕规范 + 色调统一方案

## 三个核心机制

### 1. 弹窗引导系统（AskUserQuestion）

信息不足时不乱猜，改为弹出结构化选择菜单。触发时机：

- 风格方向未指定（弹出：赛博朋克 / 国风新中式 / 写实摄影 / 3D渲染 / 扁平插画 / 水彩手绘）
- 画面比例未说明（弹出：16:9 / 9:16 / 1:1 / 4:3）
- 视频节奏未定义（弹出：史诗慢动作 / 动感快剪 / 平滑运镜 / 固定镜头）
- 用户说「再优化一下」「感觉差一点」→ 弹出诊断菜单，精准定位不满意的维度

**无限迭代**：只要你不满意，每次都触发追问，精准调整单一维度，其他维度保持不变。

---

### 2. 记忆轴系统（Memory）

每次完成出图后，自动将你的偏好写入本地记忆轴：

| 记忆轴 | 文件 | 记录内容 |
| :--- | :--- | :--- |
| 绘图偏好轴 | `memory/image_patterns.md` | 常用画风、平台偏好、渲染引擎、高频风格词 |
| 视频偏好轴 | `memory/video_patterns.md` | 常用运镜节奏、分镜转场偏好、视频平台 |
| 海报/UI 模式 | `memory/poster_ui_patterns.md` | 海报结构偏好、品牌色调规律 |
| 生图经验库 | `memory/image_prompts.md` | 历史高质量提示词归档 |

**效果**：用的次数越多，它越懂你的审美——不用每次重新解释风格方向。

你也可以主动蒸馏：把收集到的提示词技巧、小红书截图描述发过来，说「帮我提取规则存入记忆库」，AI 会压缩成结构化模式写入对应轴文件。

---

### 3. 多平台适配引擎

根据目标平台，自动切换提示词语言和格式：

| 平台 | 语言 | 格式风格 | 特殊处理 |
| :--- | :--- | :--- | :--- |
| **即梦 / 豆包** | 中文 | 自然语言描述 | 默认输出，未指定平台时使用 |
| **Midjourney** | 英文 | 关键词堆叠 | 自动附加 `--ar`、`--v`、`--style`、`--q` |
| **Stable Diffusion 1.5** | 英文 | Tag式，含 Negative Prompt | LoRA 权重标注 `<lora:xxx>` |
| **SDXL** | 英文 | 扩展 Tag式，更强细节描述 | 长提示词支持，自动补增强词 |
| **Flux / F1** | 英文 | 关键词重复声明 | 风格词重复3次可显著提升稳定性 |
| **DALL-E** | 中/英 | 完整描述性句子 | 无特殊参数 |
| **可灵 / 即梦视频 / Sora** | 中/英 | 分镜+镜头语言 | 含时长、运镜类型说明 |

## 安装与触发

### 安装

将 `wsc-image` 文件夹放入 Claude Code 的技能目录：

```bash
~/.claude/skills/wsc-image/
```

### 触发关键词

说以下任意一种，技能自动激活：

```
帮我生成一张图 / AI 画图 / AI 绘画
做个视频提示词 / 做个分镜
参考这张图 / 参考这个风格
提示词优化 / 再优化一下
帮我修图 / 调一下色 / 电影感
换背景 / 换场景 / 保留人物换场景
帮我 P 图 / 合成一张图
```
