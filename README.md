# wsc-image — AI 视觉创意专家

> 把你脑子里模糊的画面，变成 AI 能精准执行的提示词。

不管是「画一张赛博朋克风海报」还是「帮我把这张照片改成电影感」，直接说需求就行——不用背命令，不用懂提示词语法。

---

## 作用与定位（Why — 解决什么问题）

大多数人用 AI 画图，卡在「不知道怎么描述」这一步。说太少，出来的图不对；说太多，AI 理解不了。wsc-image 解决的就是这个问题：

1. **生图提示词** — 你说一句话，它给你一段经过 7 层结构优化的专业提示词，能直接用
2. **视频分镜** — 描述一个场景或故事，输出完整分镜表和每个镜头的运镜提示词
3. **参考图分析** — 上传一张你喜欢的图，它从风格、色调、光影、构图四个维度拆解，再问你想保留哪些
4. **修图/P图** — 提供原图，说「换背景」「保留人物」「电影感重建」，自动生成保护五官比例的修改指令

---

## 四大场景逻辑（How it works — 判断流程）

AI 会根据你说的话自动判断场景，不需要你手动指定：

| 用户输入情况 | 触发判定 | 执行场景 |
| :--- | :--- | :--- |
| **有原图** + 说「修图/换背景/保留人物」 | → | **场景 D**：修图/P图 |
| **有原图** + 说「参考这个风格/类似这个」| → | **场景 C**：参考图分析 |
| **无原图** + 描述静态画面（人物/产品/场景）| → | **场景 A**：AI 绘图 |
| **无原图** + 描述动态内容（运镜/视频/故事）| → | **场景 B**：视频分镜 |

**信息不够时**，会自动弹出选择题引导你（风格 / 比例 / 节奏），选一下就继续，不用你自己背命令。

**结果不满意时**，说「差一点」「再优化」「不够电影感」，会追问具体哪里不对，精准调整那一层，其他不动。**没有轮数限制。**

## 怎么使用（Step-by-step）

实操 3 步：

### 1. 安装（Claude Code）

把 `wsc-image` 文件夹放进 skills 目录：

```bash
~/.claude/skills/wsc-image/
```

### 2. 触发

直接说需求，不用记命令：

| 触发示例（原话） | 进入哪个场景 |
| :--- | :--- |
| 「帮我画一张赛博朋克风的城市夜景」 | 场景 A：AI 绘图 — 弹窗问比例和风格细节 |
| 「做一个产品发布的视频提示词」 | 场景 B：视频分镜 — 生成分镜表 |
| 「参考这张图，画一个类似氛围的人像」 | 场景 C：参考图分析 — 四维分析 + 意图确认 |
| 「把这张图的背景换成洱海，保留人物」 | 场景 D：修图/P图 — 弹窗确认保留范围 |

### 3. 迭代

说「差一点」「再优化」可无限追问：

```text
你：差点意思，光影不够强
AI：【需要调整哪里？】
    A. 主体描述  B. 色彩/光影  C. 风格方向  D. 构图  E. 氛围  F. 其他
你：B
AI：[输出增强光影版本]
```

## 如何把内容喂给 AI 并保存（蒸馏工作流）

用户核心场景：收集一批链接/文案/MD文件 → 发给 AI → AI 帮梳理提炼 → 保存为结构化知识。如果你收集了一批提示词技巧或风格参考，流程如下：

### Step 1：准备素材

把你收集到的内容整理好，可以是：
- 复制的文章/文案文本
- Markdown 文件内容
- 提示词样例（如 `修图P图.md` 里的那些）

### Step 2：发给 AI 并说明意图

用这类指令喂进去：

```text
以下是我收集的关于 [主题] 的内容，请帮我：
1. 提取核心规则和使用逻辑
2. 整理成结构化的 Markdown 笔记
3. 保存到 memory/image_patterns.md（或我指定的路径）
```

### Step 3：AI 蒸馏输出

AI 会将散乱内容压缩为：
- **规则层**：触发条件 → 执行逻辑 → 输出格式的清晰结构
- 可直接写入 `memory/` 目录下的 .md 文件长期保存
- 下次对话加载即可还原这套能力

**这套流程同样适用于扩展 wsc-image：**
把你找到的新提示词技巧、平台更新等发给 AI，让它蒸馏后补充进 `memory/image_patterns.md`。

## 平台支持

| 平台 | 语言 | 默认 |
| :--- | :--- | :--- |
| 即梦 4.0 / 4.5 | 中文自然语言 | ✅ 默认且仅输出此中文版本 |
| 豆包 | 中文自然语言 | 指定后输出 |
| Midjourney | 英文 + `--ar --v --q` 参数 | 指定后输出 |
| Stable Diffusion SDXL | 英文，双编码器优化 | 指定后输出 |
| Stable Diffusion 1.5 | 英文 Tag 堆叠 + 负向提示词 | 指定后输出 |
| 可灵 / 即梦视频 / Sora | 中/英，镜头语言 | 指定后输出 |

