这个文件只用于**生图场景**。当用户需求命中图片 / 海报 / KV / 插画 / 图标 / 概念图等内容时，优先参考本文件。

## 一、常见生图类型

### 1. 3D 图标 / Icon 组
**结构公式：**
[图标数量+排列方式] + [整体风格定义] + [材质/质感] + [光效/底座] + [背景] + [每个图标内容清单] + [调性关键词] + [渲染器/技术参数]

**分层描述法（英文提示词常用）：**
- 前景（foreground）：主体人物/产品
- 中景（center/middle）：核心视觉元素
- 背景（background）：环境氛围
- 地面（ground/floor）：地面材质和反光

## 二、平台偏好特征

| 特征 | 说明 | 示例 |
|------|------|------|
| 渲染器声明 | 明确指定渲染工具 | Blender、C4D、OC渲染 |
| 分辨率强调 | 高清参数常有帮助 | 8K、4K、超清、高保真 |
| 光效细节 | 对光的描述越细越容易出质感 | 柔和高光、体积光、边缘发光、电影级光影、光影流动、弥散渐变 |
| 构图说明 | 明确主体位置和镜头感 | 中心构图、俯视、远景、特写、留白、等轴侧视角 |
| 材质表达 | 对物体质感要具体 | 金属、玻璃、毛玻璃、流体、珠光、亚克力、水晶质感、通透材质 |
| 调性补强 | 用于统一审美方向 | 科技感、未来感、高级感、前卫、震撼、清新雅致、梦幻 |
| 色彩方案 | 明确的色彩搭配 | 蓝色系、蓝白渐变、低饱和度色调、靛紫炭黑色 |
| 风格定义 | 具体的设计风格 | 弥散风格、极简风格、清新可爱、科技感 |

## 三、生图输出建议

当用户只给一句模糊描述时，优先补全这些维度：
1. 主体是什么
2. 用途是什么（海报 / KV / 插画 / 图标 / 概念图）
3. 风格是什么
4. 构图是什么
5. 光影与材质怎么表现
6. 色调和氛围是什么
7. 是否需要比例 / 清晰度 / 渲染器说明
8. 应用场景是什么

---

## 四、使用规则
- 本文件只服务于**生图场景**
- 生图提示词输出时，最终结果必须放入代码块中，方便复制
- 优先生成一条高质量主提示词，必要时再补增强词或替代方向
- 不要机械堆砌形容词，要优先保证主体、风格、构图、光影、材质清晰
- 根据用户需求选择合适的风格和技术参数
- 结合最新的设计趋势，如弥散渐变、通透材质等元素
