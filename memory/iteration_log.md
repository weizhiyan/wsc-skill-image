## 2026-04-02 迭代记录

**场景：** 图片 — 夏日活动海报
**原始需求：** 骑车的男孩，夏天，活动页，愉快清爽的氛围
**迭代轮次：** 3轮

**每轮追问收集：**
- 第1轮：图片类型/风格/色调 → 海报KV、极简清新、冷色调
- 第2轮：主体细节/问题定位 → 光影氛围更强烈、主体不够突出
- 第3轮：核心问题/期望方向 → 用户指出风格完全不对，想要3D黏土风格 + 更活泼有动感

**问题分析：**
- 首次提示词缺失了：画面风格类型（写实/插画/3D/黏土）这个最关键的维度
- 用户最在意的维度是：风格类型 > 动感表现 > 光影氛围

**优化总结：**
- 下次同类需求应优先追问：画面风格类型（写实摄影/插画/3D渲染/黏土/像素等）
- 应在第一轮弹窗加入风格类型选项，不能只问「艺术风格」这种模糊维度
- 活动页/海报类需求要额外问：是否有特定画面风格偏好

## 2026-04-22 迭代记录

**场景：** 修图/调色 — 风景照 + 宠物照（布偶猫）
**平台偏好：** 豆包 / 即梦（中文平台），未提及Midjourney
**迭代轮次：** 风景3轮，宠物4轮

**用户偏好总结：**
- 偏好**暗调高对比**风格，不喜欢均匀明亮的日系风
- 风景类：重视**三层纵深**和**体积光落点**，光影要有戏剧张力
- 人像/宠物类：偏好**影棚硬光**而非柔补光，明暗比要高（约1:3~1:4）
- 对"氛围感"要求高，色调要有情绪温度（冷暖分离明确）
- 不喜欢提示词字数多但不精准，偏好**工业级精准描述**

**关键反馈记录：**
- "光影差点意思，没有突出高级感，纵深感不够" → 触发三层纵深框架
- "阴影对比不够明显" → 改为伦勃朗硬光，明暗比1:4
- "整体色调氛围没有" → 补充冷暖分离色调，阴影要"有颜色的黑"
- "布偶猫深色的地方不够深" → 高白调下深色区域不能提亮，保留品种特征对比

**布偶猫特殊规则（本次验证）：**
- 高白调风格下：白色区域提亮，深色斑纹区域**严格保留深度，不提亮**
- 眼睛是唯一高饱和锚点，蓝色虹膜需单独强化
- 阴影色调用近中性深灰（带轻微冷倾向），不要过饱和蓝色滤镜
